# Introduction

Welcome to the Maple Game Engine guide!

Maple is a 3D game engine written in Rust with a focus on simplicity and ease of use. This guide will teach you everything you need to know to start building 3D games and applications with Maple.

## What is Maple?

Maple is a node-based 3D game engine that provides:

- **Simple API** - Easy-to-use builder pattern for creating scenes
- **Physics Integration** - Built-in physics with Rapier
- **Event System** - Behavior-driven programming with `.on()` handlers
- **PBR Rendering** - Modern physically-based rendering
- **GLTF Support** - Load 3D models from industry-standard formats

## Who is this guide for?

This guide assumes basic familiarity with Rust programming. If you're new to Rust, we recommend reading [The Rust Book](https://doc.rust-lang.org/book/) first.

## How to use this guide

The guide is organized in a progressive manner:

1. Start with [Getting Started](getting-started.md) to create your first app
2. Learn about [Scenes](scenes.md) and how to structure your game world
3. Explore [3D Rendering](3d-rendering.md) to add visual content
4. Add interactivity with [Behaviors](behavior.md)
5. Implement realistic movement with [Physics](physics.md)
6. See it all come together in the [Player Controller Example](examples/player-controller.md)
